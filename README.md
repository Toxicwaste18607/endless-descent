# endless-descent

Endless Descent
Overview

Endless Descent is a top-down dungeon crawler built using Python and Pygame. The player navigates through a maze-like environment, fights enemies, and survives as long as possible. The project focuses on core game systems such as movement, collision, enemy AI, and combat mechanics.

Features
Player movement with sprint (stamina-based)
Collision system with walls
Enemy AI with basic pathfinding and aggression range
Combat system (player attack + enemy damage)
Health and stamina UI bars
Respawn system with random safe location
Ghost system (creates a ghost on player death)
Tile-based dungeon layout using wall objects
Controls
W / A / S / D → Move player
Left Shift → Sprint (uses stamina)
Space → Attack
Close Window → Exit game


AI use 
Ai was used as both a teacher and a debugger. However it would make several mistakes or over complicate things.
